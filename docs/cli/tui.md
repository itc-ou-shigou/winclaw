# `winclaw tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
winclaw tui
winclaw tui --url ws://127.0.0.1:18789 --token <token>
winclaw tui --session main --deliver
```
